# Performance-Engineering-Analysis

## Overview:
This project analyses cycling training data to explore how training load, intensity and physiological responses evolve over an 8 week structured training period. The aim is to identify trends in performance adaptation using real activity data exported from a fitness tracking platform.

## Project Objectives:
Develop a python-based analysis tool to investigate physiological and performance trends at the start of a structured endurance plan
Analyse how training load relates to changes in fitness and fatigue
Identify performance trends over time in cycling activites

## Skills:

- Python
- Panda
- NumPy
- Data Analysis
- Data Visualisation
- Engineering Reporting


## Planned Features: 
-	Import Garmin activity data and intervals.icu metrics
-	Clean the data automatically
-	Generate graphs automatically
-	Calculate summary statistics
-	Generate data-centric insights
